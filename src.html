src.i18n.install_language(lang_code)[source]

Install the specified language for gettext translations.

Parameters:

lang_code (str) – Language code to install (e.g., ‘en’, ‘ua’).

Side Effects:

Sets the global translation for the application. Installs the ‘_’ function in the builtins for runtime translation. Falls back to English if the specified translation is not found.
